Greek Edu Remix
===============

The Greek Edu Remix is an effort for delivering [Fedora][fedora] to Greek schools.

Maintained by the [Fedora Greek Team][fedora-el].


[fedora]: http://fedoraproject.org/
[fedora-el]: http://el.fedoracommunity.org/
